# 🏠 Документация Лендинга "Центр Международной Недвижимости"

## 📁 Структура проекта (Feature-Sliced Design)

```
src/
├── app/                    # Next.js App Router
│   ├── page.tsx           # Главная страница (композиция виджетов)
│   ├── layout.tsx         # Root layout
│   └── globals.css        # Глобальные стили
│
├── widgets/               # Виджеты (крупные самостоятельные блоки)
│   ├── Header.tsx        # Шапка с логотипом, контактами, CTA
│   ├── Hero.tsx          # Главный экран с заголовком и статистикой
│   ├── Partners.tsx      # Секция с бонусами от партнеров
│   ├── Calculator.tsx    # Форма расчета стоимости квартиры
│   ├── HowWeWork.tsx     # Секция "Как мы работаем" (6 шагов)
│   ├── Comparison.tsx    # Сравнение "Самостоятельно vs С нами"
│   ├── Map.tsx           # Карта с адресом
│   └── Footer.tsx        # Подвал с контактами и кнопкой "Наверх"
│
└── shared/               # Переиспользуемые элементы
    ├── config/
    │   └── colors.ts     # ⚡ ЦВЕТОВАЯ ПАЛИТРА (главный файл для изменения цветов)
    └── ui/               # UI-компоненты
        ├── Button.tsx    # Кнопка с вариантами (primary/accent)
        ├── Card.tsx      # Карточка
        └── Container.tsx # Контейнер для контента
```

## 🎨 Как изменить цвета проекта

**Файл**: `src/shared/config/colors.ts`

### Группы цветов:

1. **primary** - Основной фиолетовый (кнопки, акценты)
   - `main: '#7C3AED'` - основной фиолетовый
   - `dark: '#5B21B6'` - темный (hover)
   - `light: '#A78BFA'` - светлый

2. **accent** - Акцентный красный (CTA кнопки)
   - `red: '#EF4444'` - основной красный
   - `redDark: '#DC2626'` - темный (hover)

3. **background** - Фоновые цвета
   - `main: '#FFFFFF'` - белый
   - `gray: '#F3F4F6'` - светло-серый
   - `dark: '#1F2937'` - темный
   - `purple: '#4C1D95'` - фиолетовый (footer)

4. **text** - Цвета текста
   - `primary: '#111827'` - основной черный
   - `secondary: '#6B7280'` - серый
   - `white: '#FFFFFF'` - белый
   - `muted: '#9CA3AF'` - приглушенный

5. **border** - Границы
   - `light: '#E5E7EB'`
   - `gray: '#D1D5DB'`

### Пример изменения цветов:

```typescript
// Измените в colors.ts:
primary: {
  main: '#FF6B00', // Новый оранжевый
}

// Автоматически изменится во всех кнопках и акцентах
```

## 🧩 Секции лендинга

1. **Header** - Липкая шапка с логотипом, наградой, графиком работы, кнопкой консультации
2. **Hero** - Главный экран с гарантией продажи, статистикой (90%, 10 000+ семей), CTA кнопками
3. **Partners** - 4 карточки с бонусами от партнеров (скидки 40%, 30%, 10%, 20%)
4. **Calculator** - Форма с выбором состояния квартиры (6 вариантов с картинками)
5. **HowWeWork** - 6 этапов работы с клиентом
6. **Comparison** - Сравнительная таблица "Самостоятельно" vs "С нами"
7. **Map** - Яндекс карта с адресом офиса
8. **Footer** - Контакты, адрес, кнопка "Наверх"

## 🔧 Модульность

Каждый виджет **полностью независим** и может быть:
- Перемещен в другой проект
- Изменен без влияния на другие виджеты
- Удален или заменен

### Пример переноса виджета:

```typescript
// Просто скопируйте файл и импортируйте
import { Hero } from '@/widgets/Hero';
```
